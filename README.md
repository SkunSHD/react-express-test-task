## React, Express, Bootstrap, React-Data-Grid

### How to start this thing?

#### Dev mode:
###### git clone https://github.com/SkunSHD/react-express-test-task.git
###### yarn start
###### yarn server
###### http://localhost:3000/


#### Prod mode:
###### git clone https://github.com/SkunSHD/react-express-test-task.git
###### yarn build
###### yarn server
###### http://localhost:8080
